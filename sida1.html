Variabler och typer

En variabel i JavaScript deklareras med ett Variable Statement, vilket består av ordet 'var' följt av ett namn.

Ett statement som deklarerar en variabel vid namn 'foo':
var foo;

En variabel i JavaScript är otypad, men det värde som variabeln för tillfället refererar till är typat och har en av följande typer:

  • Boolean
  • Null
  • Undefined
  • Number
  • String
  • Object

Operatorn "typeof" ger en textsträng som beskriver typen för den variabel man anv-nder den på. Se specen för mer detaljer.

Vi ska gå igenom typerna lite hastigt och kommer i senare slides gå in lite mer på djupet av typen Object

Boolean

Kan anta värdena true och false. Här passar vi även på att visa kommentars-syntaxen.

Att tilldela variabler boolean values:
var foo = true;
var bar = false;
var fooType = typeof foo; // 'boolean'

Null

Kan endast anta värdet null.

Att sätta en variabeln till null:
var foo = null;

Undefined

Kan endast anta värdet undefined. Alla variabler har värdet undefined innan de tilldelas något annat värde. Det finns fler situationer där man träffar på undefined, men vi kommer till det senare.

Man tilldelar generellt inte värdet undefined till en variabel.

Variabler är undefined tills man tilldelat dem ett värde:
var foo; // undefined

Number

Ett 64 bits tal, enligt IEEE 754. D.v.s. ett gäng flyttal, samt NaN, -Inf och +Inf.

Man kan konvertera en textsträng till ett Number genom parseFloat (Eller parseInt, om man bara vill ha heltal).

Number values:
var foo = 2;
var bar = 3.14;
var baz = parseInt('1337');

String

En textsträng skapas i JavaScript genom antingen enkla eller dubbla citattecken.

En serie av UTF-16 code points när den innehåller text, men kan även innehålla godtycklig binär data.

Man kan läsa ut strängens längd m.h.a. propertien ".length". Vi återkommer till vad properties är under Object.

Man kan konkatenera strängar med +.

Skapa textsträngar:
var foo = 'Hello';
var bar = "Hello there";
var baz = "Hello" + " there";

Object

Ett JavaScript Object är en samling properties. En property är ett namn associerat med ett värde. Värdet kan vara av vilken typ som helst, inklusive i sin tur ett Object.

Det finns flera olika sätt att skapa Objects, här visas några av dem (Alla Object i exemplet är tomma objekt, utan några programdefinierade properties).

Skapa Objects:
var foo = new Object();
var bar = Object.create(null);
var baz = {};
