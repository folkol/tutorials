Variabler och typer

En variabel i JavaScript deklareras med ett variable-statement, vilket består av ordet 'var' följt av en inom nuvarande scope unikt namn. Vad ett scope är går vi igenom senare.

Ett statement som deklarerar en variabel vid namn 'foo':
var foo;

En variabel i JavaScript är inte typad, men det värde som variablen refererar till har en av följande typer:

  • Boolean
  • Null
  • Undefined
  • Number
  • String
  • Object
